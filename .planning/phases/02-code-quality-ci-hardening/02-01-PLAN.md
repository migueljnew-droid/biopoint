    **Step 1: Create `apps/api/src/types/fastify.d.ts`**

    Create the FastifyRequest augmentation file using declaration merging. This is the official Fastify TypeScript pattern (https://fastify.dev/docs/latest/Reference/TypeScript/).

    ```typescript
    declare module 'fastify' {
      interface FastifyRequest {
        // Set by middleware/auth.ts
        userId: string;
        userEmail: string;
        userRole: 'USER' | 'ADMIN';

        // Set by app.ts onRequest hook
        startTime: number;

        // Set by middleware/automaticLogoff.ts
        sessionId?: string;
        sessionTimeout?: number;

        // Set by middleware/rateLimit.ts
        rateLimit?: {
          limit: number;
          remaining: number;
          resetTime: number;
        };
        accountLockout?: {
          locked: boolean;
          lockedUntil?: Date;
          failedAttempts: number;
        };
      }
    }

    export {};
    ```

    CRITICAL: `request.id` and `request.log` are ALREADY typed on FastifyRequest by Fastify itself (Fastify sets request.id via genReqId, request.log is a Pino logger). Do NOT add them to the declaration — just remove the `as any` wrapper at usage sites.
