## Memory System (CROSS-IDE SYNC)

### To Save Memories (Syncs to ALL IDEs automatically)

Write to `.council/memories.jsonl`:

```json
{{"timestamp": "ISO8601", "content": "what you learned", "category": "lesson", "source": "antigravity"}}
```

### One-liner to save a memory:

```bash
echo '{{"timestamp": "'$(date -u +%Y-%m-%dT%H:%M:%SZ)'", "content": "YOUR_MEMORY_HERE", "category": "lesson", "source": "antigravity"}}' >> .council/memories.jsonl
```

### Categories:
- `lesson` - Applies everywhere
- `preference` - User preference
- `context` - Project-specific
- `task` - Task completed/TODO

The Council Daemon automatically syncs to central memory.
